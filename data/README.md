Directory to store raw JSON responses from the API.

Files are named accordingly:

| Name     | File name format       |
|----------|------------------------|
| Classes  | `classes*.json`        |
| Courses  | `courses*.json`        |
| Grades   | `student-grades*.json` |
| Students | `students*.json`       |
| Users    | `users*.json`          |

where `*` is a wildcard.
